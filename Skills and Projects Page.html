Highlighted Projects

Automated Data Pipeline — Collins Aerospace (2025)

Designed and implemented end-to-end automated data pipelines using Python and SQL to replace manual data entry and reporting processes across engineering teams. The solution reduced manual reporting time by over 40% and significantly increased data reliability. Using error handling and logging mechanisms, the pipeline maintained a high level of data integrity.

Final dashboards were integrated with reporting tools and shared with stakeholders to enable real-time decision-making. The insights generated helped leadership make data-driven capital investment decisions and prioritize long-term reliability strategies for high-value assets.

Tools Used: Python, SQL, Excel, SharePoint, Power BI

Power BI Dashboard & Workflow Automation — Collins Aerospace (2024)

Collaborated with operations and engineering teams to design interactive Power BI dashboards that provided live insights into bottlenecks in production processes. Through DAX formulas and custom visuals, the dashboards revealed key areas of delay, enabling the company to optimize work cell layouts and reduce process cycle time.

Additionally, built multiple workflow automations using Power Automate and custom JSON logic, eliminating redundant tasks like status updates and manual task assignments. This improved efficiency and accuracy across departments while cutting down administrative workload.

Tools Used: Power BI, Power Automate, Excel, JSON, Teams, SharePoint

Pricing Quality Improvement Project — Honeywell Aerospace (2023)

Spearheaded a pricing quality project to detect and eliminate inconsistencies in over 200,000 part numbers across 7 catalog tables. Using Power BI, I designed a data model that uncovered more than 6,000 pricing errors, enabling leadership to take corrective action.

To prevent recurring issues, I created a PowerApps tool that allowed team members to enter, validate, and approve pricing data through a clean and structured UI. This drastically reduced file redundancy and improved pricing data quality across the board.

Tools Used: Power BI, PowerApps, Excel, SQL, SharePoint

Manufacturing System Design Project — MC Green Solutions (2023)

As part of my final undergraduate capstone, I worked with a real-world manufacturing client, MC Green Solutions, to analyze and improve their plant layout and system performance. I used MATLAB simulations to model different layout options and identify areas of congestion, imbalance, and inefficiency.

After running several scenarios, I proposed a new facility layout using AutoCAD that would reduce average haulage distances by 52%. The design was presented to and approved by plant leadership and has since been considered as part of their operational expansion plans.

Tools Used: AutoCAD, MATLAB, Excel, SketchUp
